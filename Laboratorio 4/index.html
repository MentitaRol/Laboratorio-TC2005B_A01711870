¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Solo hay similitudes muy superficiales entre Java y JavaScript como las puede haber con otros lengujaes de programación como su sintaxis y algunos conceptos de programación orientada a objetos, sin embargo los dos lenguajes están diseñados para resolver problemas muy diferentes.

¿Cómo se declara una variable con alcance local dentro de una función?

Declarando las variables con 'let' solo estaran disponibles dentro de nuestras funciones y no se puede acceder fuera de ellas.
'let nombre de variable'

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  • Hacen que nuestro codigo no sea seguro y facil de vulnerar
  • Disminuye la legibilidad
  • pueden provocar un comportamiento inesperado
  • Dificultan la modularidad del código.

Laboratorio 5: Frameworks de estilo

Describe Material design

MMaterial es un sistema de diseño creado por Google para ayudar a los equipos a crear experiencias digitales de alta calidad para Android, iOS, Flutter y la web. Proporciona un conjunto de principios, directrices y componentes para crear interfaces de usuario visualmente atractivas y altamente funcionales. Se inspira en el mundo físico y sus texturas, combinando principios de diseño clásico con tecnologías digitales avanzadas para lograr una experiencia intuitiva y coherente en diferentes dispositivos y plataformas.

Material Design - Version 2. (s. f.). Material Design. https://m2.material.io/design/introduction
