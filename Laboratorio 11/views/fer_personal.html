Laboratorio 12: HTML dinámico

¿Qué otros templating engines existen para node?

Un motor de plantillas es una tecnología que permite separar el marcado HTML de los datos dinámicos. Simplifica el proceso de representación de contenido dinámico al proporcionar marcadores de posición para los datos dentro de las plantillas HTML.
